Bankruptcy Analysis Tool

About

This web app provides tools to help you assess the bankruptcy risk of your business. Harnessing the power of machine learning, the Predict tool analyzes 95 different business parameters and returns a value indicating your bankruptcy risk. In development of the machine learning model, we were able to achieve an accuracy rating of 94.79% in predicting bankruptcy status on data that was not used during the training phase.

Through the Data Dash, this app also provides several useful data visualizations based on the data set used to develop the machine learning model. Using the Data Dash in addition to the machine learning prediction algorithm can help you gain valuable insights into the health of your business financials.
